# Student Registration Form

## 📌 Project Overview

The Student Registration Form is a responsive web application built using HTML and CSS. It allows users to enter their personal and educational details through a clean and user-friendly interface. This project demonstrates the fundamentals of HTML forms and modern CSS styling.

---

## 🚀 Features

- Responsive user interface
- Personal information section
- Educational details section
- Gender selection using radio buttons
- Skills selection using checkboxes
- Course selection using dropdown
- Password and confirm password fields
- Register and Reset buttons
- Attractive gradient background
- Modern card layout with shadow effects

---

## 🛠 Technologies Used

- HTML5
- CSS3
